Nastavení editoru

Editor je základní pomůcka každého programátora, takže je dobré do něj investovat trochu času.

Je víceméně jedno který editor budeš používat; pokud už nějaký oblíbený máš, stačí ho jen nastavit. Pokud ale používáš Poznámkový blok (Notepad) z Windows, nebo editor předinstalovaný v Mac OS X, nebude ti stačit.
Pro Windows si můžeš stáhnout Notepad++, pro Mac mi byl doporučen Atom.
Pro Linux doporučuji Kate (instalace na Fedoře: sudo yum install kate, na Ubuntu/Debianu: sudo apt-get install kate).

Správný editor umí kromě editování a ukládání textu tři základní věci popsané níže – buď přímo, nebo po nastavení.

Existují i složitější a mocnější editory, takzvané IDE (Integrated Development Environment, integrované vojové prostředí), třeba PyCharm, Eclipse nebo KDevelop. IDE umí spoustu pokročilých funkcí jako našeptávání, přejmenovávání, spouštění programů a podobně. Na začátek nejsou moc vhodné. Chceš-li takový editor přesto použít, měla bys přesně vědět, co za tebe dělá editor, a jak to spravit až něco udělá špatně.

Návody jsou jen pro editory, které mám k dispozici; ostatní by měly být podobné (a návod pro ně rád doplním).

Číslování řádků

Nalevo před každým řádkem se ukazuje číslo. Tohle se bude velice hodit, až Python bude nadávat, že chyba je na řádku 183.

  • Kate: menu Pohled/View → Ukazovat čísla řádek/Show Line Numbers
  • Gedit: Menu (Upravit/Edit) → Předvolby/Preferences → Zobrazit/View → Zobrazovat čísla řádků/Display Line Numbers
  • Notepad++: není potřeba nastavovat

Odsazování

Stisknutím klávesy Tab na začatku řádku se vloží 4 mezery. Pro psaní a sdílení kódu v Pythonu je důležité, aby byly čtyři, a aby to byly opravdu mezery. (Jestli to jsou mezery se dá zjistit šipkou ←; pokud se kurzor pohne o čtyři pozice najednou, nejsou to mezery).
Stisknutím Shift+Tab se o čtyři mezery „od-odsadí“.

  • Kate: menu Nastavení/Settings → Nastavit 'Kate'/Configure Kate → Úpravy/Editing → Odsazování/Indentation
    • Výchozí režim odsazení/Default indentation mode: Python
    • Odsazovat pomocí/Indent using: Mezer/Spaces
    • Šířka tabulátoru/Tab Width: 4 znaky
    • Odsadit pomocí/Indentation width: 4 znaky
    • Klávesa Backspace zpětně odsazuje v úvodních mezerách/Backspace key in leading blank space unindents
  • Gedit: Menu (Upravit/Edit) → Předvolby/Preferences → Editor
    • Šířka tabulátorů/Tab width: 4
    • Vkládat mezery místo tabulátorů/Insert spaces instead of tabs
    • Povolit automatické odsazování/Enable automatic indentation
  • Notepad++: Nastavení → Předvolby → Nastavení tabulátoru/Tab Settings → Zaměnit za mezery/Replace by Space

Zvýrazňovaní syntaxe

Barvou textu editor napovídá, jak bude programu Python rozumět. Třeba řetězec "Hello world!" v prvním programu by měl být zobrazen jinou barvou.

Barvičky se naukládají do souboru, text se obarvuje jen na základě toho, co v něm je. Kamarádka s jiným editorem může mít stejný soubor obarvený docela jinak.

  • Kate: automaticky (po uložení souboru s koncovkou .py)
  • Gedit: automaticky (po uložení souboru s koncovkou .py)
  • Notepad++: automaticky (po uložení souboru s koncovkou .py)

A další

Editor může navíc třeba automaticky odsazovat, když předchozí řádek končí dvojtečkou, nebo sám ukončovat závorky. Nauč se psát se svým editorem, a jestli ti něco nevyhovuje, zkus to v nastavení vypnout.
