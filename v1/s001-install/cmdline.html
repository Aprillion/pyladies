Příkazová řádka

Nejdříve musíme otevřít příkazovou řádku – to černé okýnko, které programátoři používají na zadávání příkazů. Na první pohled může vypadat nepřirozeně, ale dá se na ni zvyknout :)

Příkazová řádka (respektive program, kterému se říká i konzole či terminál) se na různých systémech otevírá různě:

  • Windows (cs): Start → Všechny programy → Příslušenství → Příkazový řádek
  • Windows (en): Start → All Programs → Accessories → Command Prompt
  • Mac OS X (en): Applications → Utilities → Terminal
  • Linux (KDE): Hlavní Menu → hledat Konsole
  • Linux (Gnome): Super → hledat Terminál

Nevíš-li si rady, zkus buď googlit, nebo se jednoduše zeptat (na sraze pomocníka, jinak e-mailem).

Po otevření konzole tě uvítá řádek, kterým počítač vybízí k zadání příkazu. Podle systému bude končit buď znakem $ nebo >, před nímž můžou být ještě další informace:

Unix (Linux, OS X)

$

Windows

>

První příkaz

Začneme jednoduchým příkazem. Napiš whoami (z angl. who am I? – kdo jsem?) a stiskni Enter. Objeví se přihlašovací jméno. U mě to vypadá takhle:

Unix

$ whoami
pviktori

Windows

> whoami
pviktori

Aktuální adresář

Příkazová řádka pracuje vždy v nějakém adresáři. ("Adresář" je to samé co "složka".) Který to je, nám poví příkaz pwd nebo cd (z angl. print working directory – vypiš pracovní adresář, resp. current directory – aktuální adresář).

Unix

$ pwd
/home/pviktori/

Windows

> cd
C:\Users\pviktori
Aktuální adresář se většinou ukazuje před znakem $ nebo >, ale je dobré pwd znát, kdyby ses náhodou ztratila.

Co v tom adresáři je?

Příkaz ls nebo dir (z angl. list – seznam; directory – adresář) nám vypíše, co aktuální adresář obsahuje – všechny soubory, včetně podadresářů, které se v aktuálním adresáři nacházejí.

Unix

$ ls
Applications
Desktop
Downloads
Music
...

Windows

> dir
 Directory of C:\Users\pviktori
05/08/2014 07:28 PM <DIR>      Applications
05/08/2014 07:28 PM <DIR>      Desktop
05/08/2014 07:28 PM <DIR>      Downloads
05/08/2014 07:28 PM <DIR>      Music
...

Změna aktuálního adresáře

Aktuální adresář se dá změnit pomocí příkazu cd (z angl. change directory – změnit adresář). Za cd se píše jméno adresáře, kam chceme přejít – pokud máš adresář Desktop nebo Plocha, přejdi tam. Pak nezapomeň ověřit, že jsi na správném místě.

Unix

$ cd Desktop

$ pwd
/home/pviktori/Desktop
pozn. Unix rozeznává u adresářů a souborů velikost písmen: Desktop a desktop není to samé.

Windows

(cd jsme už viděly; ale předtím jsme nepsaly nic za něj) 
> cd Desktop

> cd
C:\Users\pviktori\Desktop
pozn. Pokud přecházíš do adresáře na jiném disku, například D: místo C:, je potřeba kromě cd zadat jméno disku jako zvláštní příkaz (např. D:).

Vytvoření adresáře

Co takhle si vytvořit adresář na PyLadies? To se dělá příkazem mkdir (z angl. make directory – vytvořit adresář).

Unix

$ mkdir pyladies

Windows

> mkdir pyladies

Úkol

Zkus v nově vytvořeném adresáři pyladies vytvořit adresář test, a zkontrolovat že se opravdu vytvořil.
Budou se hodit příkazy cd, mkdir, ls či dir.

Řešení

Unix

$ cd pyladies
$ mkdir test
$ ls
test

Windows

> cd pyladies
> mkdir test
> dir
05/08/2014 07:28 PM <DIR>      test

Úklid

Teď vytvořené adresáře zase smažeme. Nemůžeme ale smazat adresář, ve kterém jsme, takže se vrátíme na Desktop. Nadřazený adresář (ten, který obsahuje ten aktuální), se značí dvěma tečkami:

Unix

$ cd ..
$ pwd
/home/pviktori/Desktop

Windows

> cd ..
> cd
/home/pviktori/Desktop
A nakonec smažeme vytvořený adresář pyladies. K tomu použijeme příkaz rm nebo rmdir (z remove – odstraň, resp. remove directory – odstraň adresář). Pozor, příkazová řádka nepoužívá odpadkový koš! Všechno se nadobro smaže, takže si dobře překontroluj, že mažeš správný adresář.

Unix

$ pwd
/home/pviktori/Desktop
$ rm -rv pyladies
removed directory: ‘pyladies’

Windows

> cd
/home/pviktori/Desktop
> rmdir /S pyladies
pyladies, Are you sure ? Y

Konec

A to je vše! Můžeš příkazovou řádku zavřít. To se dělá příkazem exit.

Unix

$ exit

Windows

> exit
